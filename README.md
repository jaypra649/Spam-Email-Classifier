# Spam Email Classifier

This project uses Machine Learning to classify messages as Spam or Ham.

## Technologies Used

- Scikit-learn
- TF-IDF Vectorizer
- Naive Bayes Classifier

## Features
- Text classification
- Spam detection
- Machine learning prediction

2. Paste the code
3. Run all cells
